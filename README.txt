QUICK TEST
1. Unzip the folder.
2. Open index.html in Chrome or Safari.
3. Enter a weight or repetitions.
4. Refresh the page. The information should remain saved.

GOOGLE SITES
Google Sites does not run uploaded HTML/JavaScript directly. Host this folder on a static
web host such as GitHub Pages, Netlify, or Cloudflare Pages, then use Google Sites:
Insert > Embed > URL.

PHONE USE
After hosting, open the site in Safari and select Share > Add to Home Screen.

DATA
Workout entries are saved only in that browser on that device using localStorage.
Clearing browser website data will erase the saved workout entries.
